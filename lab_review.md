# Lab Report – M-Shot Prompting

## Objective

The objective of this lab was to explore how Zero-shot, One-shot, and Few-shot prompting influence the quality, consistency, and formatting of responses generated by a Large Language Model (LLM). Three different topics were tested: Colombian politics, FIFA World Cup statistics, and the Spanish La Liga championship.

## Experiment 1 – Colombian Politics

In the first experiment, I created prompts related to the Presidents of Colombia using Zero-shot, One-shot, and Few-shot prompting.

The Zero-shot prompt produced a correct answer but without any specific formatting. After providing one example, the model learned to return the answer using the same structure (President and Political Party). When two examples were provided, the model consistently maintained the requested format for new questions.

## Experiment 2 – FIFA World Cup

The second experiment focused on the top goal scorers in different FIFA World Cup tournaments.

Again, the Zero-shot prompt generated correct answers but included additional information. The One-shot prompt successfully taught the model to answer using only the requested fields (Player and Team). After extending the prompt to a Few-shot example by adding the number of goals, the model correctly generalized the format to other World Cup editions, producing well-structured and consistent responses.

This experiment demonstrated that Few-shot prompting is an effective way to control the structure of the model's output without explicitly describing the desired format.

## Experiment 3 – Spanish La Liga

The third experiment focused on La Liga champions.

The model correctly learned the requested format ("Team" and "Runner-up"), but it failed to answer one factual question correctly. When asked who won the 2013/2014 La Liga title, the model answered **Real Madrid**, while the correct answer is **Atlético Madrid**. I repeated the experiment using a temperature of 0 to reduce randomness, but the model continued producing the same incorrect answer.

This shows that Few-shot prompting helps the model learn the response format, but it does not guarantee factual accuracy. The model can still produce incorrect information even when the prompt is well designed.

## Variations That Did Not Work Well

The La Liga experiment was the main example where the model produced an incorrect factual response. Lowering the temperature did not solve the issue, indicating that the error was not caused by randomness but by the model's internal knowledge or reasoning.

Another observation was that Zero-shot prompts often produced more detailed answers than requested. One-shot and Few-shot prompting significantly improved formatting consistency by providing examples for the model to imitate.

## What I Learned

This lab demonstrated that examples are often more effective than instructions alone when controlling the structure of LLM outputs. Zero-shot prompting is useful for general questions, but One-shot and Few-shot prompting produce more consistent and predictable formatting.

I also learned that Few-shot prompting teaches the model **how** to respond rather than **what is factually correct**. Even when the output format was perfect, the model was still capable of generating incorrect information. Therefore, prompt engineering improves response quality and consistency, but factual answers should still be verified when accuracy is important.

Overall, this exercise showed that prompt engineering is a powerful technique for guiding model behavior, but it is not a substitute for fact-checking or external verification.
